Here's the full walkthrough of that flow, start to finish:
At the store (or wherever you're scanning):

Open your live Pour Index link and tap Start scanning to turn on the camera.
Tap the Quick Capture toggle so it says "On." This is what enables the fast pass — no stopping to type for every bottle.
Hold each bottle's barcode inside the amber frame.

If it's already in your shelf or the shared dictionary, it just silently adds one and shows a quick toast — no interruption.
If it's totally unknown, it snaps a photo automatically and shows "Captured — keep scanning." You don't stop moving.


Keep going down the shelf, one bottle after another.

Later, when you're ready to label them:

Scroll down to the "To label" section — it'll show a count like "12 captured."
Tap Review now.
For each one, you'll see:

The photo you took, full-size
The barcode number
A name field, pre-filled with a best-guess:

If you've added your own Anthropic API key in ⚙ Recognition settings, it asks Claude to actually read the label — usually a solid, accurate guess.
If no key is set, it falls back to free on-device text recognition (Tesseract), which is rougher, especially on stylized fonts.


A brand field and a category dropdown (also auto-guessed)


Check the guess, fix anything wrong, then tap Save to shelf — this adds it to your inventory and teaches the shared dictionary that barcode, so nobody (including you) has to look it up again.
It automatically moves to the next captured photo. Repeat until the queue's empty.
Skip punts an item to the back of the line if you want to come back to it later; Discard deletes it if the photo's unusable.

That's the whole loop — fast capture in the moment, unhurried labeling afterward.
